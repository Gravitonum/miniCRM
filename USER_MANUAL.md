# User Manual: Authentication & Organization Flow

## 1. Registration
- Access the `/register` page.
- Enter your **Username**, **Email**, and **Password**.
- Click **Register**.
- You will be assigned the default "Viewer" role and redirected to the login page.

## 2. Login
- Use your **Username** and **Password** to log in.
- The system will check if your account is active and if you are already part of an organization.

### Scenario A: New User (No Organization)
- You will be automatically redirected to the **Join Organization** page.
- You must enter a valid **Organization Code** (provided by your administrator).

### Scenario B: Returning User (With Organization)
- You will be redirected directly to the **Dashboard**.

### Scenario C: Account Disabled
- If your `isActive` status in the database is set to `false`, you will see an error message and won't be able to access the application.

## 3. Joining an Organization
- On the `/join-organization` page:
    1. Enter the **Organization Code**.
    2. Click **Lookup**.
    3. If found, the company name will be displayed.
    4. Click **Join Organization** to confirm.
- Once confirmed, your settings are saved, and you are redirected to the dashboard.

## 4. Сделки (Deals)
- Доступ к разделу **Сделки** осуществляется через левое боковое меню (иконка портфеля).
- Система предоставляет два вида отображения:
    - **Канбан-доска:** визуальное управление сделками по этапам (Первый контакт, Квалификация, КП отправлено и т.д.).
    - **Список:** табличное представление всех сделок с возможностью поиска и фильтрации.
- **Создание сделки:**
    - Нажмите кнопку **+ Новая сделка** в правом верхнем углу.
    - Заполните форму в открывшейся боковой панели (название, сумма, этап, ответственный, дедлайн).
    - Новая сделка мгновенно появится на доске или в списке.
- **Карточка сделки:**
    - Кликните на карточку или строку в списке, чтобы открыть детали.
    - Отслеживайте прогресс по этапам в верхней части экрана.
    - Переходите к следующему этапу с помощью кнопки **Следующий этап →**.
    - Просматривайте финансовые детали и связанные контакты.
- **Локализация:** весь интерфейс сделок поддерживает русский и английский языки. Переключение осуществляется через иконку глобуса в хедере.

## 5. Настройки
В разделе Настройки (доступно из бокового меню) вы можете управлять параметрами вашего рабочего пространства. Раздел разделен на 4 вкладки:

### Пользователи
Здесь отображается список всех сотрудников, привязанных к вашей компании.
- Вы можете видеть статусы пользователей (Активен / Заблокирован).
- Доступна функция блокировки/разблокировки доступа для каждого пользователя.
- Кнопка «Пригласить пользователя» предназначена для добавления новых коллег в ваше рабочее пространство.

### Профиль компании
Основные параметры вашего тенанта:
- **Название компании**: Имя организации, отображаемое в системе.
- **Часовой пояс, Валюта, Язык**: Настройки по умолчанию для всех пользователей компании.
- **API ключ Контур.Фокус**: Укажите ключ для автоматического заполнения реквизитов компаний (ИНН, ОГРН, адреса) при добавлении новых клиентов в базу.

### Справочники
Управление выпадающими списками в карточках клиентов:
- **Источники лидов**: (Сайт, Телеграм, Рекомендация и др.) — аналитика каналов продаж.
- **Организационно-правовые формы (ОПФ)**: (ООО, АО, ИП) — для точной классификации юр. лиц.
- **Типы отношений**: (Клиент, Партнер, Поставщик) — метки для быстрого понимания роли компании.
- Добавляйте новые значения или удаляйте неактуальные.

### Воронки и этапы
Настройка ваших бизнес-процессов:
- Вы можете создать несколько различных воронок продаж (например, "Новые продажи", "Продление", "Отмена").
- Для каждой воронки можно настроить свой набор этапов.
- Каждый этап имеет статус завершения: В работе (открыт), Закрыто успешно, Закрыто неуспешно. Это позволяет системе корректно считать конверсию и строить отчеты.

## 6. База Клиентов и Контактов
- Раздел доступен через левое боковое меню.
- **Модуль "Клиенты"** позволяет управлять компаниями, с которыми вы работаете.
    - Карточка клиента содержит 4 вкладки: "Основное", "Контакты", "Сделки", "История".
    - Интеграция с Контур.Фокус: при добавлении ИНН клиента можно автоматически заполнить его реквизиты.
- **Модуль "Контактные лица"** служит для ведения истории общения с конкретными людьми.
    - Один контакт может быть привязан к нескольким компаниям.
    - При добавлении нового контакта можно указать его должность и контакты для связи.

## 7. Дашборд
- Основной экран системы (Dashboard) предоставляет агрегированную информацию:
    - **Виджет "Мои активные сделки"** показывает количество сделок, за которые вы ответственны.
    - **Виджет "Выиграно в этом месяце"** показывает сумму успешно завершенных сделок в текущем месяце.
    - **Виджет "Дедлайны на этой неделе"** обращает внимание на сделки, срок которых истекает в ближайшие дни.
    - График **"Воронка продаж"** визуализирует распределение активных сделок по текущим этапам воронки.

## 8. Импорт и Экспорт данных
- Раздел доступен через левое боковое меню по кнопке "Импорт / Экспорт".
- **Импорт:**
    - Позволяет массово загружать Клиентов и Сделки из CSV и Excel (XLSX) файлов.
    - Вы можете просто перетащить файл в зону загрузки (Drag & Drop) или выбрать его через диалоговое окно.
    - Поддерживаются два режима: создание новых записей и обновление существующих (для клиентов – по совпадению ИНН, для сделок – по названию).
    - После загрузки отобразится предпросмотр из 5 строк для удобного сопоставления (маппинга) колонок файла с полями системы.
    - После успешного импорта вы увидите подробный отчет о загруженных строках и возможных ошибках валидации данных.
- **Экспорт:**
    - Данные можно выгрузить в удобном табличном виде, нажав на кнопку "Экспорт" на страницах **Клиенты** и **Сделки**.
    - База будет автоматически сохранена на ваш компьютер в формате Excel (.xlsx), с учетом текущих примененных фильтров и поиска.
