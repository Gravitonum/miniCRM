# GraviSales CRM

Мини-CRM система для автоматизации B2B-продаж, построенная на платформе **GraviBase**.

## Основные возможности
- **Управление сделками:** Канбан-доска и табличный вид.
- **Мультитенантность:** Каждая компания работает в своем изолированном пространстве.
- **Мультиязычность:** Полная поддержка русского и английского языков.
- **Темная и светлая темы:** Комфортная работа в любое время суток.
- **Адаптивный интерфейс:** Современный дизайн на базе Tailwind CSS v4.

## Технологический стек
- **Frontend:** React + Vite + TypeScript.
- **Styling:** Tailwind CSS + Lucide Icons.
- **Backend:** GraviBase (Backend-as-a-Service).
- **i18n:** react-i18next.

## Начало работы
1. Склонируйте репозиторий.
2. Перейдите в папку `frontend`.
3. Выполните `npm install`.
4. Запустите проект: `npm run dev`.
