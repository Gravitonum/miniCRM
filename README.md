# GraviSales CRM

Мини-CRM система для автоматизации B2B-продаж, построенная на платформе **GraviBase**.

## Основные возможности
- **Управление сделками:** Канбан-доска и табличный вид.
- **Мультитенантность:** Каждая компания работает в своем изолированном пространстве.
- **Интеграция с Контур-Фокус:** Автозаполнение реквизитов компаний по ИНН.
- **СуперАдмин панель:** Мониторинг всех организаций и управление их статусом (блокировка).
- **PWA (Progressive Web App):** Установка приложения на устройства, поддержка работы оффлайн.
- **Интерактивный тур:** Обучающий тур для новых пользователей по функционалу Дашборда.
- **Кроссплатформенность и Локализация:** Полная поддержка русского и английского языков, темная и светлая темы, адаптивный дизайн (Tailwind CSS v4).

## Технологический стек
- **Frontend:** React + Vite + TypeScript.
- **Styling:** Tailwind CSS + Lucide Icons.
- **Backend:** GraviBase (Backend-as-a-Service).
- **i18n:** react-i18next.

## Начало работы
1. Склонируйте репозиторий.
2. Перейдите в папку `frontend`.
3. Выполните `npm install`.
4. Запустите проект: `npm run dev`.
